I’m working on a talk for an upcoming conference and decided I’d try and set up an RStudio Cloud project to work it from my iPad1. I was able to clone the repository wtihout any issue, but ran into an authentication issue when I tried to commit and push my changes back up. I knew I’d run into this issue before, so I went looking for solutions and found my answer in the wonderful Happy Git and GitHub for the useR.

The problem was I’d cloned the repository as HTTPS, but because my GitHub account is secured with 2FA, I needed to connect via SSH.

  1. Generate a public key using GitHub
  2. Add the public key to GitHub
  3. Switch remote URLs from HTTPS to SSH

Update

Ironically, I wasn’t the only one who ran into this issue recently.

I finally added 2-factor authentication to GitHub, but it broke my setup with #RStudio. But I was able to figure it out (SSH) thanks to #happygitwithR #rstats https://t.co/hxPsOQpPLw Thanks @JennyBryan @jimhester_ et al!
